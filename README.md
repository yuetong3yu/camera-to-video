# Camera to Web

Clone the project to your local:

```bash
git clone https://github.com/yuetong3yu/camera-to-video.git
cd camera-to-video
```

install dependenies(most of them are developing depdencies, no worry, it won't pack into your build version):

```bash
yarn
```

or

```bash
npm install
```

Then Just Run it:

```bash
yarn start
```

or

```bash
npm start
```
